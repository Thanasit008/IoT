# 🖱️ UI_CONTROLS.md — ปุ่มและฟังก์ชันที่ผู้ใช้กดได้ในระบบ

## 1. หน้า Local Mode (ตั้งค่าเครื่องครั้งแรก)

เข้าถึงผ่าน `http://192.168.4.1` — เสิร์ฟจากตัว ESP8266 เอง ไม่ต้องมีอินเทอร์เน็ต

### 1.1 ตั้งค่า WiFi ของเครื่อง (Device AP)

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |
|---|---|---|
| ช่องชื่อ WiFi ของเครื่อง (SSID) | ช่องกรอกข้อความ | ตั้ง/แก้ไขชื่อ WiFi ที่เครื่องปล่อยออกมา |
| ช่องรหัสผ่าน WiFi ของเครื่อง | ช่องกรอกรหัสผ่าน | ตั้ง/แก้ไขรหัสผ่านของ WiFi เครื่อง |
| ปุ่ม "บันทึกชื่อ/รหัสผ่านเครื่อง" | ปุ่มกด | บันทึกและสั่งให้เครื่องปล่อย WiFi ตามค่าใหม่ |
| ปุ่ม "รีเซ็ตค่าเริ่มต้นของเครื่อง" | ปุ่มกด | ล้างชื่อ/รหัสผ่าน AP กลับเป็นค่าโรงงาน |

### 1.2 ตั้งค่า WiFi ที่จะให้เครื่องเชื่อมต่อ (Home WiFi)

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |
|---|---|---|
| ปุ่ม "สแกนหา WiFi" | ปุ่มกด | ค้นหารายชื่อ WiFi ใกล้เคียง |
| ช่อง SSID WiFi บ้าน | ช่องกรอก/เลือกจากรายการ | เลือกหรือพิมพ์ชื่อ WiFi ที่ต้องการเชื่อมต่อ |
| ช่องรหัสผ่าน WiFi บ้าน | ช่องกรอกรหัสผ่าน | กรอกรหัสผ่านของ WiFi บ้าน |
| ปุ่ม "บันทึก / เชื่อมต่อ" | ปุ่มกด | สั่งเครื่องเชื่อมต่อ WiFi บ้านและ HiveMQ |
| ปุ่ม "ลืมเครือข่ายนี้" | ปุ่มกด | ลบ WiFi บ้านที่เคยบันทึกไว้ |

## 2. หน้ากรอก Device ID + Password (แทนหน้า Login)

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |
|---|---|---|
| ช่อง Device ID | ช่องกรอกข้อความ | กรอกรหัสประจำเครื่องที่ต้องการคุม |
| ช่อง Password | ช่องกรอกรหัสผ่าน | กรอกรหัสผ่านของเครื่องนั้น |
| ปุ่ม "สแกน QR Code" | ปุ่มกด (เปิดกล้อง) | กรอก Device ID โดยสแกน QR แทนการพิมพ์ |
| ปุ่ม "เข้าใช้งาน" | ปุ่มกด | ส่งไปตรวจสอบผ่าน Cloud Function ถ้าถูกต้องจะได้ Token และเข้าสู่ Dashboard |
| ช่องติ๊ก "จำเครื่องนี้ไว้" | Checkbox | เลือกว่าจะบันทึก Device ID ไว้ใน localStorage เพื่อไม่ต้องกรอกซ้ำ |

## 3. หน้าเพิ่ม/สลับเครื่อง

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |

| รายการเครื่องที่เคยกรอกไว้ | รายการเลือก | สลับไปคุมเครื่องอื่นที่เคยกรอก Device ID ไว้ในเบราว์เซอร์นี้ |
| ปุ่ม "เพิ่มเครื่องใหม่" | ปุ่มกด | เปิดหน้ากรอก Device ID + Password เพื่อเพิ่มเครื่องอื่น |
| ปุ่ม "ลืมเครื่องนี้" | ปุ่มกด | ลบเครื่องออกจากรายการในเบราว์เซอร์ (ไม่ใช่ลบเครื่องจริง) |

## 6. ส่วนตั้งเวลาให้อาหารอัตโนมัติ (สูงสุด 4 รอบ/วัน)

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |
|---|---|---|
| ช่องเลือกเวลา (แต่ละรอบ) | Time Picker | กำหนดเวลาที่ต้องการให้อาหารอัตโนมัติ |
| ช่องกรอกปริมาณอาหาร (แต่ละรอบ) | ช่องกรอกตัวเลข | กำหนดจำนวนกรัมของแต่ละรอบ |
| สวิตช์เปิด/ปิดรอบ (แต่ละรอบ) | Toggle Switch | เปิด/ปิดใช้งานรอบนั้นโดยไม่ต้องลบทิ้ง |
| ปุ่ม "เพิ่มรอบให้อาหาร" | ปุ่มกด | เพิ่มรอบใหม่ (ได้สูงสุด 4 รอบ) |
| ปุ่ม "ลบรอบ" | ปุ่มกด | ลบรอบให้อาหารที่ไม่ต้องการ |
| ปุ่ม "บันทึกการตั้งค่า" | ปุ่มกด | บันทึกตารางเวลาลง Firebase |

## 7. ส่วนระบบชั่งน้ำหนัก

| องค์ประกอบ | ประเภท | ผู้ใช้ทำอะไรได้ |
|---|---|---|
| ปุ่ม "Tare (ปรับศูนย์)" | ปุ่มกด | ตั้งค่าน้ำหนักปัจจุบันให้เป็นศูนย์ |
| ปุ่ม "Calibration (ปรับเทียบค่า)" | ปุ่มกด + กรอกน้ำหนักอ้างอิง | ปรับเทียบความแม่นยำของ Load Cell |
| ช่องกรอกน้ำหนักมาตรฐาน | ช่องกรอกตัวเลข | ใช้ระบุน้ำหนักจริงตอน Calibration |
| ช่องกรอก "ปริมาณอาหารที่ใช้ต่อวัน" | ช่องกรอกตัวเลข | ใช้คำนวณจำนวนวันอาหารคงเหลือ |
| ปุ่ม "บันทึกค่า Calibration" | ปุ่มกด | บันทึกค่าที่ปรับเทียบแล้วลง Firebase |
